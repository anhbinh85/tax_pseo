# Push to GitHub (Step by Step)

Repository: `https://github.com/anhbinh85/tax_pseo`

## 1) Initialize Git (if not already)
```bash
git init
```

## 2) Add the remote
```bash
git remote add origin https://github.com/anhbinh85/tax_pseo
```

## 3) Check status
```bash
git status
```

## 4) Stage files
```bash
git add .
```

## 5) Commit
```bash
git commit -m "Initial commit"
```

## 6) Push to GitHub
```bash
git branch -M main
git push -u origin main
```

## 7) Verify
- Open the repo in your browser and confirm files are visible.
